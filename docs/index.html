Features

Not a chatbot hack.
A real memory engine.

Every session, your AI starts knowing exactly what it knew last time. Decisions, bugs, architecture, config — all searchable in under 5ms. No re-explaining. No wasted tokens. Ever.

🔍
Hybrid FTS5 + Vector Search
Keyword matching via SQLite FTS5 fused with optional sqlite-vec vector similarity using Reciprocal Rank Fusion. Fast, accurate, zero cloud dependency.
// Find anything, instantly
brain_search("Stripe webhook idempotency")
// → ranked results in <5ms
🧬
Biological Memory Decay
Memories fade like the Ebbinghaus curve. Unused ones lose relevance. Critical ones never decay. Access resets the clock.

📂
Project Scoping
Context stays isolated per project. Your SaaS memories don't pollute your CLI tool context.

⚡
Smart Session Lifecycle
Auto-detect your project from the workspace path. Load the right context at session start. Extract new facts at session end with v1.1 Auto-Learning.
brain_session_start_smart — detects project, loads top 25 memories
You work — agent auto-stores decisions, bugs, architecture
brain_session_end — extracts facts, stores them permanently
🔒
100% Local & Private
Everything lives in ~/.mcp-ai-brain/brain.db on your machine. No cloud sync. No API calls. No telemetry. Your code stays yours.
